Plexus Compiler

Plexus Compiler is a Plexus component to use different compilers through a uniform API.

It is composed by:

  • plexus-compiler-api: the API to use compilers,
  • plexus-compiler-manager: a manager to choose a compiler,
  • plexus-compilers: different compilers
    • plexus-compiler-aspectj: AspectJ compiler,
    • plexus-compiler-csharp: C#/Mono compiler,
    • plexus-compiler-eclipse: Eclipse compiler,
    • plexus-compiler-javac: javac compiler,
    • plexus-compiler-javac-errorprone: javac compiler with error-prone static analysis checks enabled ,
    • plexus-compiler-jikes: jikes compiler,
  • plexus-compiler-test: a test harness.
